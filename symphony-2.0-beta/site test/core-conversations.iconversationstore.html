IConversationStore interface

Signature
export interface IConversationStore extends IStore<IConversation> 
export declare namespace IConversationStore

Methods

MethodDescription
addMember(conversationId, userId, init)

Add a member to a conversation

The current user must be a member of the conversation to be able to add another member, and unless the membersCanAddUsers setting is true, the current user must also be an owner.

createGroupChat(members)

Create a new group chat

The initial members will be the current user and all users given in the members argument.

createIM(userId)

Create a one to one chat with another user

If an IM conversation with the given user already exists, that same conversation will be returned

createRoom(name, description, members, settings)

Create a new room

The initial members will be the current user and all users given in the members aegument.

getConversation(conversationId)

Get a conversation

Only conversations that are public, or that the acting user is a member of, are available. So even if the specified conversation ID actually exists, this method will fail with Missing Link if the conversation is not available to the acting user.

getConversations()

Gets all conversations that the acting user is a member of

The returned array of conversations will be sorted according to the time the latest message in the conversation is sent, in descending order. Conversations without any messages will be sorted after conversations that do.

getMember(conversationId, userId)

Get member information

getMembers(conversationId)

Get members of a conversation

hideConversation(conversationId)

Hide a conversation

joinConversation(conversationId)

Join a conversation

The conversation must be a public room.

leaveConversation(conversationId)
muteConversation(conversationId)

Mute a conversation

All notifications for incoming messages will be muted.

unhideConversation(conversationId)

Unhide a conversation

unmuteConversation(conversationId)

Unute a conversation

All notifications for incoming messages will be unmuted.

updateConversation(conversationId, settings)

Update the settings of a conversation.

updateMember(conversationId, userId, update)

Update a member of a conversation

This can be used for example to promote a member to an owner, or to demote an owner to a regular member.
